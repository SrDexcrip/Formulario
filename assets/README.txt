Instrucciones para la carpeta assets/

Coloca aquí las imágenes que quieras usar en las tarjetas de la galería. Nombres recomendados:
- proyecto1.jpg (1200x675 px, ratio 16:9)
- proyecto2.jpg
- proyecto3.jpg
- formulario.jpg
- proyecto5.jpg
- proyecto6.jpg

Formato recomendado: JPG o WebP (optimizado). Si necesitas transparencia usa PNG.
Optimiza el peso (ej. TinyPNG, Squoosh) para mejorar tiempos de carga.

Si no se encuentra una imagen, la página mostrará un placeholder automático.
